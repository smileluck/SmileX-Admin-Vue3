# SmileX-Admin

A Admin Managerment Platform by Vue3


# 技术栈
- 基础框架：Vue3
- 路由：vue-router
- css框架：sass
- UI框架：element-plus
- http请求：axios
- 状态管理：pinia
- 编辑器：vditor
